Computational Fluid Dynamics

从物理问题到
可信的流场结果

CFD（计算流体力学）通过数值方法求解流体控制方程， 用于预测流场中的速度、压力、湍流特性，以及升力、阻力、压降等工程参数。

1 定义计算目标、工况与输出变量
2 建立几何模型和流体计算域
3 进行网格划分和边界层加密

6 后处理、网格验证和实验对比

Workflow

CFD 计算的六个关键步骤

一个可靠的 CFD 项目不只是“运行软件”，还需要合理的物理假设、网格、 边界条件及计算结果验证。

STEP 01

明确计算目标

确定研究对象和输出量，例如压力分布、速度场、升力系数 CL、 阻力系数 CD、压降、传热或流动分离位置。

目标与工况
STEP 02

建立物理模型

判断流动是二维还是三维、稳态还是瞬态、层流还是湍流、 可压缩还是不可压缩，并选择适当的湍流模型。

控制方程与假设
STEP 03

建立几何与计算域

清理不影响流动的微小结构，保留关键几何特征； 在模型外建立足够大的流体域，减小远场边界干扰。

CAD 与计算域
STEP 04

划分网格

将连续流场离散为有限数量的网格单元。 在壁面、前缘、后缘、分离区和尾流区域进行局部加密。

空间离散
STEP 05

设置边界并求解

设置入口速度、出口压力、壁面无滑移条件和湍流参数， 选择求解器、离散格式及压力—速度耦合算法。

数值求解
STEP 06

后处理与验证

输出云图、流线、压力系数和气动力结果；通过残差、 网格无关性和实验数据对比判断计算结果是否可信。

验证与确认

Governing Equations

CFD 求解的核心方程

CFD 基于质量守恒、动量守恒和能量守恒，将连续控制方程离散为代数方程， 再由计算机进行迭代求解。

质量守恒方程

∂ρ/∂t + ∇ · (ρu) = 0

质量守恒表示流体质量不会凭空产生或消失。对于低速不可压缩流， 可以近似为 ∇ · u = 0。

动量守恒方程

ρ(∂u/∂t + u · ∇u) = −∇p + μ∇²u + ρg

Navier–Stokes 方程描述压力、黏性力、惯性力和重力共同作用下的流动状态。

雷诺数

Re = ρUL / μ

雷诺数表示惯性力与黏性力的相对大小。翼型计算中， 雷诺数直接影响边界层、转捩和分离现象。

升力与阻力系数

CL = L / (½ρU²A)   |   CD = D / (½ρU²A)

CL 和 CD 是衡量翼型气动性能的重要无量纲参数， 可用于不同工况和不同翼型之间的对比。

Quality Control

如何判断 CFD 结果可靠？

软件计算结束或残差下降，并不意味着结果一定正确。 还需要从以下方面检查数值计算的可信度。

✓

残差与目标量收敛

除了残差，还应观察 CL、CD、压力、压降或质量流量是否趋于稳定。

✓

质量守恒检查

检查入口和出口质量流量是否基本平衡，避免出现明显的守恒误差。

✓

网格无关性分析

比较粗、中、细网格下的结果，确认继续加密后 CL、CD 等关键结果变化较小。

✓

近壁网格与 y⁺

第一层网格高度应与壁面处理方法和湍流模型要求相匹配。

✓

计算域大小检查

入口、出口和远场边界要远离翼型，避免边界条件影响翼型周围流场。

✓

实验与参考数据验证

将 Cp、CL、CD 与公开实验数据、风洞结果或可信参考结果进行比较。

Case Study

NACA 4412 孤立翼型 CFD 计算案例

本课程案例以 NACA 4412 二维孤立翼型为对象，进行低速外流场数值模拟， 分析不同攻角下翼型的流场、压力分布、升力和阻力。

NACA 4412 Airfoil

NACA 4412 翼型外流场分析

NACA 4412 是经典四位数翼型：最大弯度为翼弦长度的 4%， 最大弯度位置位于 40% 弦长处，最大厚度为翼弦长度的 12%。

学生应以视频为参考，完成几何建立、外流场构造、网格划分、 边界条件设置、求解计算和后处理，并与实验或公开数据进行比较。

计算对象： NACA 4412 二维孤立翼型，以翼弦长度 c 作为特征长度。
推荐计算域： 入口距离前缘约 5c，出口距离后缘约 10c～20c，上下边界距离翼型不小于 5c。
推荐边界条件： 入口采用速度入口，出口采用压力出口，翼型表面设置为无滑移壁面。
推荐模型： 低速不可压缩流、二维稳态 RANS；湍流模型可优先选择 SST k-ω。
网格重点： 前缘、后缘、翼面边界层和下游尾流区域局部加密，并检查 y⁺ 分布。
主要输出： Cp 分布、速度云图、压力云图、流线、CL、CD、CL/CD 和分离位置。
计算项目 建议设置 说明
计算类型 二维、稳态、外流场 适合基础翼型平均气动力性能分析。
流体模型 低速不可压缩空气 普通翼型风速通常远低于声速，可近似不可压缩。
湍流模型 SST k-ω 通常适用于近壁流动、逆压梯度和分离预测。
建议攻角 0°、4°、8° 可自行扩展至 −4°～16°，分析失速前后的性能变化。
网格验证 粗、中、细三套网格 比较 CL、CD、Cp 等关键结果，证明网格无关性。
结果对比 CFD 与实验/参考数据 保证雷诺数、攻角、参考面积及参考速度定义一致。

Course Assignment

课程任务：复现视频并完成 CFD 与实验对比报告

学生需独立完成 NACA 4412 孤立翼型二维 CFD 计算， 复现视频中的基本工作流程，并完成数值结果与实验/公开参考结果的对比分析。

一、核心任务要求

  • 观看 NACA 4412 翼型 CFD 案例视频并复现其基本工作流程。
  • 使用 AirfoilTools 获取或核对 NACA 4412 翼型几何坐标与参考数据。
  • 建立 NACA 4412 二维孤立翼型外流场计算模型。
  • 完成流体域、翼型壁面、前缘、后缘和尾流区域的合理网格划分。
  • 设置合适的边界条件、湍流模型、求解器和收敛判据。
  • 至少完成 3 个不同攻角下的数值计算。
  • 完成 CFD 计算结果与实验/公开数据的对比报告。

二、建议计算工况

  • 翼型：NACA 4412 二维孤立翼型。
  • 计算方法：二维稳态 RANS 数值模拟。
  • 流体介质：低速不可压缩空气。
  • 湍流模型：SST k-ω；条件允许时可研究转捩模型影响。
  • 攻角设置：建议至少计算 0°、4°、8° 三个工况。
  • 扩展任务：可计算更大攻角，分析翼型接近失速及失速后的流动特征。
  • 网格要求：至少构建粗、中、细三套网格进行无关性分析。

三、计算与实验结果对比

  • 从 AirfoilTools 或其他公开可靠来源获取 NACA 4412 参考数据。
  • 对比前确认雷诺数、攻角、来流速度、参考面积定义保持一致。
  • 比较不同攻角下的升力系数 CL。
  • 比较不同攻角下的阻力系数 CD。
  • 绘制并比较翼型上下表面的压力系数 Cp 分布。
  • 说明 CFD 与实验数据之间的误差，并分析可能原因。
  • 误差分析可从网格、湍流模型、转捩、二维假设和实验条件等角度展开。

四、提交报告内容

  • 研究目标、NACA 4412 翼型参数和计算工况说明。
  • 几何模型、计算域尺寸、边界条件和物理模型说明。
  • 网格策略、网格数量、边界层设置、y⁺ 分布和网格无关性分析。
  • 求解器设置、残差收敛曲线及 CL、CD 监测曲线。
  • 速度云图、压力云图、流线图和尾流分析结果。
  • Cp、CL、CD、CL/CD 等数值计算结果。
  • CFD 结果与实验/公开参考数据的对比图表及误差分析。
  • 最终结论、不足之处和后续改进建议。
